# 📄 Muhammed Furkan Demirci | Resume-as-Code Project

Bu repository, akademik ve profesyonel geçmişimi **JSON tabanlı bir veri yapısı** ve **otomatikleştirilmiş iş akışları** ile sunan bir projedir.

## 🏗️ Proje Mimarisi
- **Data Source:** `resume.json` dosyasında tüm bilgilerim tek bir merkezde toplanmıştır.
- **Multi-language Support:** Türkçe, İngilizce ve Fransızca dillerinde Markdown dökümanları hazırlanmıştır.
- **CI/CD Pipeline:** GitHub Actions kullanılarak, her güncellemede otomatik PDF çıktısı üretilmektedir.

- **Boğaziçi Üniversitesi:** Mühendislik Temelleri (2021 - 2024)
